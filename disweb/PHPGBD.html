Existen varios gestores de base de datos y de igual forma son varios que son compatibles para interactuar con PHP pero el mas más conocido es el gestor MySQL que es de código abierto.


MYSQL es un gestor de base de datos relacional, multihilo y multiusuario.


Relacional: Quiere decir que la información esta relacionada en si misma a través de varias tablas.


multihilo: La base de datos soporta varios procesos a la vez es decir varias consultas, peticiones de manera simultánea.


Multiusuario: Puede haber varios usuarios conectados simultáneos a la base de datos.


Es importante conocer el lenguaje MySQL para poder realizar consultas, insertar, eliminar o actualizar datos en la base de datos y de esta manera poder complementar con PHP.
