PHP viene del acrónimo recursivo de “Hypertext Preprocesador” especialmente adecuado en el desarrollo web y puede ser incrustado en HTML. Este es un lenguaje de código abierto, es decir es gratuito y libre siendo posible utilizar por todos los programadores.


El desarrollar aplicaciones web con PHP permite integrar de manera rápida y eficiente la interacción con otros lenguajes de programación obteniéndose como resultado aplicaciones dinámicas.


Grandes empresas han desarrollado sus aplicaciones con este lenguaje de programación y a sido vinculado con importantes gestores de bases de datos para aprovechar al máximo sus ventajas y se mas competitivos en el mercado teniendo en cuenta la eficiencia y rapidez que se puede acceder a los datos para su manipulación en cuanto a consulta, ingresar, eliminar o actualizar registros.
